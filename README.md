<b>Instruções:</b> 
- Use a barra de pesquisa para encontrar o seu Pokemon favorito, pelo número ou pelo nome.
- Use o botão 'prev' para voltar ao Pokemon anterior.
- Use o botão 'next' para avançar ao próximo Pokemon.
